Abstract

Multi-turn jailbreaks are a serious challenge for large language models because successful attacks can unfold gradually across an extended conversation. Prior approaches usually fall into two camps: template-based methods rely on fixed prompt sequences that cannot adapt to the target model's replies, while optimization-based methods react to responses but still optimize mostly for immediate success without explicit long-horizon planning.

Multi-Turn Jailbreaker introduces a training framework for building attack agents that reason over dialogue history, revise plans across turns, and choose the next move with explicit strategic intent. The pipeline first expands diverse tree-structured attack trajectories, then initializes the agent with supervised fine-tuning, and finally applies TreeRPO, a reinforcement learning variant that uses tree structure to improve policy optimization under sparse delayed rewards.

On subsets of AdvBench, HarmBench, and HarmfulQA, the resulting agent reaches the strongest average attack success rate among compared baselines and shows a clear advantage on stronger and safety-aligned target models.

Why It Matters

Reasoning Over History

The agent explicitly analyzes past turns instead of treating each prompt independently, which makes it better suited for long conversations.

Planning, Not Just Reacting

Each action includes structured reasoning plus the next prompt, allowing the attacker to revise plans as the dialogue evolves.

Tree-Guided Learning

Tree expansion exposes intermediate branches with downstream potential, which improves data efficiency compared with learning only from terminal outcomes.

Method Overview

1

Jailbreak Forests

Start from harmful seed prompts and expand a dialogue tree with strategy templates, intermediate reasoning, target responses, and judge rewards.

2

Cold Start SFT

Use curated tree traces to teach the attacker how to format actions, apply multi-turn strategies, and reason over conversation state.

3

TreeRPO

Fine-tune the policy with reinforcement learning that groups sibling nodes and uses leaf-guided rewards to handle sparse delayed feedback.

Main Contributions

Agent-based formulation

The paper formalizes multi-turn jailbreak as an interaction between an attack agent, a target model, and a judge, making the state, action, and reward structure explicit.

End-to-end training pipeline

The method combines tree-based data generation, supervised fine-tuning, and reinforcement learning into a practical recipe for long-horizon jailbreak training.

Leaf-guided optimization

TreeRPO makes better use of tree-structured rollouts by optimizing over sibling branches and intermediate trajectories instead of only terminal nodes.

Strong empirical performance

The approach consistently improves attack success across open-source and commercial targets, with particularly clear gains on aligned models.
